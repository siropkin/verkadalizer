# Verkadalizer — Menu Visualization for Verkada

Make your workplace menu feel instantly more appetizing. Verkadalizer enhances menu images on Verkada Menu pages with photoreal food visuals, customized to your dietary preferences and presented in distinct visual styles.

Click the ✨ button next to any menu image, then toggle between the original and the generated visualization anytime.

---

## Quick Start

1. Install the extension and open its settings
2. Connect an AI provider by adding your API key
3. Visit the Verkada Menu page
4. Click ✨ next to a menu image to generate a visualization
5. Switch between Original / Generated with one click

---

## What You’ll Love

- **Photoreal food visualizations**: Clean, professional-looking food imagery designed to complement the menu layout
- **Diet-aware dish selection**: Picks dishes that fit common dietary preferences (e.g., vegetarian, vegan, gluten-free, keto, etc.)
- **Distinct visual styles**: Choose from multiple bold looks (from classic tabletop editorial to modern floating studio layouts)
- **Optional translation**: Translate menu text into supported languages while preserving layout
- **Smooth in-page controls**: Generate, stop/cancel, and toggle views without leaving the page
- **Progress feedback + caching**: Clear status updates while processing; generated images can persist across reloads

---

## Privacy & Security

- **Your API key stays local**: Stored only in your browser’s local storage
- **No analytics / tracking**: No external data collection by the extension
- **Works only where it should**: Runs on Verkada Menu pages

---

## Requirements

- Access to the Verkada Menu page: `https://sites.google.com/verkada.com/verkada-menu`
- An API key for a supported AI provider (configured in the extension settings)
